## Prepare
1. install snarkjs
```shell
npm install -g snarkjs@0.5.0
```

2. install circom 2.1.2, more refer to https://docs.circom.io/getting-started/installation/
```shell
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```
